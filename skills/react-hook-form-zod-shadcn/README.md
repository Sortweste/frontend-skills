# React Hook Form + Zod + shadcn/ui

Build forms in React using react-hook-form for form state management, Zod for schema validation, and shadcn/ui for accessible form components.

## Installation

```bash
npx skills add https://github.com/Sortweste/frontend-skills --skill react-hook-form-zod-shadcn
```

## Packages

The following packages must be installed for using this skill

```bash
npm install react-hook-form@7.71.1 zod@4.3.6 @hookform/resolvers@5.2.2
```

## Tech Stack

This skill was designed for the following technology versions:

- **Next.js**: 16.0.10
- **React**: 19.2.1
- **React DOM**: 19.2.1
- **TypeScript**: ^5
